Git commands

1. Write name

git config --global user.name"write name here"

You can change the text (write name here) with name which you want to give to your file.

2.

git config --global user.email "your-email@example.com"

You can add your email by using this command.

3. Important command

git status : -s

The (git status -s) command is a short/summary version of git status that shows a compact overview of your Git repository's changes.

4. Intitialize repositories

git init

You can Intitialize your repositories with this command

5. Stage all files

git add

Only stage changed files

git add -A

Select all files

git add -u

Select all files included changed and deleted only execpt of new files

6. Massege and commit

git commit -m "write messege here"

  1. Stages all tracked changes if you ran (git add first).
  2. Creates a commit with your message.
  3. Records the snapshot in your local repository

7. See branch

git branch

By using this command you can see the type of this command

8. Create branch

git branch :branch name:

You can Create branch by using this command

9. Go into branch

git checkout :branch name:

You can go into branch by using this command

10. Merge branch

git merge branch.name

You can merge your branch by this command

11. Push

git push -u origen master

To open new origin

git push origen master

To push in the same origin

git push -u origen master

12. Delete branch

git branch -d [branchname]

You can delete branch with this command

13. Permanently delete branch

git push origin --delete [branch_name]

This command delete your branch permanently
